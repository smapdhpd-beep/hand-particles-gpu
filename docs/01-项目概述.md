# 项目概述

## 项目名称
hand-particles-gpu — GPU 手势粒子星云

## 灵感来源
基于 mediapipe-hand-particles 的技术资产，突破 8,000 粒子上限，探索手势控制的大规模 GPU 粒子宇宙。

## 核心体验
- 26 万粒子在手中流动
- 手作为引力奇点，扫出螺旋吸积盘
- 多种力场预设实时切换
- 粒子发光，暗背景中形成星云效果

## 当前功能
- GPUComputationRenderer 粒子物理模拟（512×512 = 262,144 粒子）
- 5 种力场预设：引力奇点 / 涡旋 / 排斥 / 布朗运动 / 简谐震荡
- MediaPipe Hands 手势输入
- 手势映射：捏合=引力、张开=涡旋、数字1=排斥、数字2=布朗、数字3=简谐
- Three.js ShaderMaterial 粒子渲染（AdditiveBlending 发光）

## 技术架构
- 渲染：Three.js + WebGL 2.0
- 物理：GPUComputationRenderer（FBO 纹理模拟位置/速度）
- 输入：MediaPipe Hands（原生 getUserMedia）
- 语言：Vanilla JS，ES Module + Import Map
