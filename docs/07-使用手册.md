# 使用手册

## 快速开始

```bash
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 手势控制

| 手势 | 效果 |
|------|------|
| 默认/放松 | 漂浮星云（弱涡旋） |
| 拇指+食指捏合 或 握拳（单手/双手） | 宇宙级黑洞：每只手可独立生成一个黑洞 |
| 五指完全张开 | 形态塑形：粒子组成爱心 / 球体 / 螺旋 / 环面，每个形态有双色渐变 |
| 比 1 | 排斥 |
| 比 2 | 布朗运动 |
| 比 3 | 简谐震荡 |

> 双手同时捏合/握拳可生成两个黑洞；双手靠近（间距约一掌宽）时合并成 8 字形黑洞，两个环之间粒子互相流动，分开后恢复两个独立黑洞。首次点击/手势会自动开启空间化音效，可用左上角 🔊 按钮静音。

### 快捷键

| 按键 | 功能 |
|------|------|
| `S` | 手动切换塑形形态 |
| `C` | 循环切换全局配色主题（星云粉 / 宇宙青 / 银河紫 / 恒星金 / 极光绿） |

### 深度交互

- **手远近推进/拉远**：手靠近摄像头，相机推进特写；手远离，相机拉远纵览
- **手远近控制速度**：手越近，粒子运动/吸收越快；手越远，越慢

## 当前状态

- 6.5 万 GPU 粒子
- 手很远或消失时，力场自动减弱，粒子保持漂移

## 最佳实践

- 手与摄像头保持 30~80cm 距离
- 确保光线充足
- 做手势时手指清晰伸直或弯曲

## 故障排查

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 画面黑屏 | 浏览器不支持 WebGL 2.0 | 换用 Chrome/Firefox/Edge |
| 巨大光斑/卡顿 | 粒子数过多或点太大 | 已优化至 6.5 万 + 6px 点大小 |
| 粒子不跟随手 | 摄像头未授权 | 检查浏览器权限 |
| 全部塌缩到中心 | 手恰好位于画面中心 | 移动手的位置 |
