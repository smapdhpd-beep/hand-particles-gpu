# 项目概述

## 项目名称
hand-particles-gpu — GPU 手势粒子星云

## 灵感来源
基于 mediapipe-hand-particles 的技术资产，突破 8,000 粒子上限，探索手势控制的大规模 GPU 粒子宇宙。

## 核心体验
- 6.5 万粒子在手中流动
- 手作为力场中心，驱动粒子形成星云、黑洞等效果
- 不同手势切换不同物理行为
- 暗背景中低调粒子构成柔和星云，黑洞以真实暗区 + 吸积盘辉光呈现

## 当前功能
- GPUComputationRenderer 粒子物理模拟（256×256 = 65,536 粒子）
- **宇宙级黑洞模式**：单手/双手捏合或握拳生成黑洞；暗事件视界、Bloom 吸积盘辉光、热辐射色彩渐变
- **双黑洞系统**：两只手各生成一个黑洞；双手靠近时合并成伯努利 8 字形黑洞，两个环之间粒子互相流动，分开后恢复两个独立黑洞
- **形态物理塑形**：五指张开进入 shape 模式，粒子被拉向爱心、球体、螺旋、环面等几何形态，每个形态有双色渐变配色，形态中心跟随主手移动
- **全局宇宙配色主题**：星云粉 / 宇宙青 / 银河紫 / 恒星金 / 极光绿，按 `C` 键切换
- **手远近驱动相机变焦**：手靠近摄像头推进特写，远离拉远纵览
- **空间化音效反馈**：深空环境嗡鸣、黑洞低频轰鸣、吸积盘流动声、模式切换提示音；声音随 hand 位置 3D 定位
- 5 种力场预设 + 形态塑形：引力奇点 / 涡旋 / 排斥 / 布朗运动 / 简谐震荡 / 形态塑形
- MediaPipe Hands 手势输入（最多 2 只手），handedness 固定左右手槽位
- 手势映射：捏合或握拳=黑洞、张开=弱涡旋漂浮、五指张开=形态塑形、数字1=排斥、数字2=布朗、数字3=简谐
- **手远近控制速度**：手靠近摄像头时旋转/吸收加快，远离时变慢
- 后期处理：UnrealBloomPass + ACES 色调映射
- 极简暗色粒子渲染：无发光拖尾，靠 Bloom 堆积出柔和星云
- 手很远/无手时自动停止力场，粒子自由漂浮

## 技术架构
- 渲染：Three.js + WebGL 2.0
- 物理：GPUComputationRenderer（FBO 双纹理：位置 + 速度）
- 输入：MediaPipe Hands（原生 getUserMedia）
- 语言：Vanilla JS，ES Module + Import Map
