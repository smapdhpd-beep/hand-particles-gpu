# 项目概述

## 项目名称
hand-particles-gpu — GPU 手势粒子星云

## 灵感来源
基于 mediapipe-hand-particles 的技术资产，突破 8,000 粒子上限，探索手势控制的大规模 GPU 粒子宇宙。

## 核心体验
- 10万~50万粒子在手中流动
- 手作为引力奇点，扫出螺旋吸积盘
- 多种力场预设实时切换
- 形态间物理流动，而非硬切

## 当前功能
- 项目骨架已搭建
- Three.js 场景就绪

## 技术架构
- 渲染：Three.js + WebGL
- 物理：GPUComputationRenderer / WebGPU
- 输入：MediaPipe Hands
- 音效：Web Audio API（待定）
