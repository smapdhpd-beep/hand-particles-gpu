# 使用手册

## 快速开始

```bash
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 手势控制

| 手势 | 力场效果 |
|------|----------|
| 默认/放松 | 涡旋 |
| 拇指+食指捏合 或 握拳 | 黑洞：约 1.2 秒缓慢成形，粒子螺旋坠入后从远处重生 |
| 比 1 | 排斥（粒子被手推开） |
| 比 2 | 布朗运动 |
| 比 3 | 简谐震荡 |

> 松开捏合/拳后，黑洞会在约 0.6 秒内逐渐消散。

## 当前状态

- 6.5 万 GPU 粒子
- 手很远或消失时，力场自动减弱，粒子保持漂移

## 最佳实践

- 手与摄像头保持 30~80cm 距离
- 确保光线充足
- 做手势时手指清晰伸直或弯曲

## 故障排查

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 画面黑屏 | 浏览器不支持 WebGL 2.0 | 换用 Chrome/Firefox/Edge |
| 巨大光斑/卡顿 | 粒子数过多或点太大 | 已优化至 6.5 万 + 6px 点大小 |
| 粒子不跟随手 | 摄像头未授权 | 检查浏览器权限 |
| 全部塌缩到中心 | 手恰好位于画面中心 | 移动手的位置 |
