# 使用手册

## 快速开始

```bash
python3 -m http.server 8080
# 浏览器打开 http://localhost:8080
```

## 手势控制

| 手势 | 力场效果 |
|------|----------|
| 默认/放松 | 涡旋 |
| 拇指+食指捏合 | 引力奇点（粒子被手吸引） |
| 比 1 | 排斥（粒子被手推开） |
| 比 2 | 布朗运动 |
| 比 3 | 简谐震荡 |

## 最佳实践

- 手与摄像头保持 30~80cm 距离
- 确保光线充足
- 做手势时手指清晰伸直或弯曲

## 故障排查

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 画面黑屏 | 浏览器不支持 WebGL 2.0 | 换用 Chrome/Firefox/Edge |
| 粒子不跟随手 | 摄像头未授权 | 检查浏览器权限 |
| 帧率低 | GPU 性能不足 | 降低窗口分辨率 |
